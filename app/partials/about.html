Update LDAP group members. Groups in LDAP must exist a priori, this service updates only members of added groups

API

Method Uri Description
GET /api/groups List groups
POST /api/groups Create new group
GET /api/groups/GROUP_NAME Get existing group
PUT /api/groups/GROUP_NAME

Sets group.

It creates a new one if group doesn't exist or updates members if group already exists. Example of body for '/api/groups/MyGroup' PUT request could be: 

          {
            name: "MyGroup",
            members: ["login1", "login2"]
          }
